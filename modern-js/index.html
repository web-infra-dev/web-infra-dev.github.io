Modern.js Framework

Rust 构建轻松切换到 Rspack 构建工具，编译飞快。一体化开发在同一项目中完成 BFF 开发，享受简洁的函数调用。嵌套路由文件系统即路由，附赠全套性能优化。多渲染模式SSR、SSG、SPR 等多种渲染模式，通通开箱即用。CSS 方案CSS Modules、CSS-in-JS、Tailwind CSS，任你挑选。易于配置以零配置启动，然后一切皆可配置。
