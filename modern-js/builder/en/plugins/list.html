Overview#

Official Plugins#

  • @modern-js/builder-plugin-vue: Used to build Vue 3 App.
  • @modern-js/builder-plugin-vue2: Used to build Vue 2 App.
  • @modern-js/builder-plugin-swc: Use SWC as the transformer and minimizer to improve build performance.
  • @modern-js/builder-plugin-stylus: Use Stylus add CSS preprocessor.
  • @modern-js/builder-plugin-esbuild: Use esbuild as the transformer and minimizer to improve build performance.
  • @modern-js/builder-plugin-node-polyfill: Inject Polyfills of Node core modules in the browser side.
  • @modern-js/builder-plugin-image-compress: Compress the image resources used in the project.

You can find the source code of these plugins in modern.js/packages/builder directory.
